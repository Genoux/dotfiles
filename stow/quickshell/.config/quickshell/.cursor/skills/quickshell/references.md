# Quickshell Local Reference

Distilled from `OVERVIEW.md` and codebase. For upstream API, use Context7 (`/websites/quickshell_v0_3_0`).

## Directory Map

```
.config/quickshell/
├── shell.qml                 # Entry: Variants → Bar per screen
├── Colors.qml                # GENERATED by matugen (not in stow)
├── bar/
│   ├── Bar.qml               # PanelWindow layout
│   └── widgets/              # 18 feature widgets
├── components/               # IconButton, Pill, InfoPill, CavaVisualizer, PrivacyButton
├── config/                   # Style, Launchers, IconRegistry (singletons)
├── services/                 # Privacy, TrayFocus, Network, Keyboard, Temperature, WeatherState, Dotfiles
└── assets/
    ├── icons/                # Bundled SVG overrides (22)
    ├── cava/config
    └── scripts/privacy-monitor.sh
```

## Module Imports

| Import | Contents |
|--------|----------|
| `qs.bar` | `Bar.qml` |

## Bar Widget Order (`bar/Bar.qml`)

**Left:** Workspaces, SystemTray  
**Center:** WindowTitle  
**Right:** PrivacyIndicator → MediaPlayer → Volume → Network → Bluetooth → ScreenRecord → Keyboard → Battery → Weather → SystemTemp → Clock → SystemInfo → SystemMenu → Dotfiles

## Service Responsibilities

| Service | Mechanism | Output |
|---------|-----------|--------|
| `Privacy` | Long-running `privacy-monitor.sh` | `webcam`, `mic`, `screenrecord`, `anyActive` |
| `TrayFocus` | Tray click handler | activate → MPRIS raise → Hyprland focus |
| `Keyboard` | `hyprctl devices -j` + IPC | Layout; toggles via `system-switch-keyboard` |
| `Network` | `ip route get 8.8.8.8` poll (5s) | Default route interface |
| `Temperature` | hwmon/nvidia-smi bash (30s) | CPU/GPU average °C |
| `WeatherState` | curl wttr.in (10 min) | Temp + condition for `$WEATHER_CITY` |
| `Dotfiles` | `FileView` on `updates.state` | `updatesAvailable` badge |

## Style Constants (`config/Style.qml`)

| Constant | Value | Used by |
|----------|-------|---------|
| `barHeight` | 38 | Bar |
| `barMargin` | 8 | Bar padding |
| `pillHeight` / `pillWidth` | 24 / 22 | Icon buttons |
| `pollIntervalFast` | 1000 ms | Keyboard fallback |
| `pollIntervalNormal` | 5000 ms | Network |
| `pollIntervalSlow` | 30000 ms | Temperature |

Fonts: SF Pro Text (sans), JetBrainsMono Nerd Font (mono/icon).

## Icon Strategy (`config/IconRegistry.qml`)

1. Check bundled override in `assets/icons/`
2. Fall back to `Quickshell.iconPath()` from Freedesktop theme
3. `IconButton` tints bundled SVGs via `ColorOverlay`

## Dependencies (Arch)

`quickshell`, `cava`, `impala`, `wiremix`

## Known Gaps vs AGS

- No brightness/volume OSD overlays in Quickshell bar
- No in-bar notifications (Mako only)
- Clock opens gnome-calendar (not calcurse)
- Bluetooth uses blueman-manager (not bluetui)
